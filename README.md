4.15 총선 - 18대~20대 총선 후보자 & 당선자 통계 
=====

4.15 총선을 앞두고 중앙선관위에 예비 후보로 등록한 391명을 분석한 결과 '50대 남성' 키워드로 요약 할 수 있었다.
18대부터 20대까지 과거 총선 후보자와 당선자의 통계를 통해 어떤 변화가 있었는지 성별, 연령, 직업, 학력을 통해 알아보자.
